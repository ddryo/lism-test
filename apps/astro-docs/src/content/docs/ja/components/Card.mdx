カード型でコンテンツを表示できるコンポーネントです。 全体をリンク化することも可能です。

```jsx
<Card>
	<CardMedia>
		<img src="..." />
	</CardMedia>
	<CardBody>
		...Content...
	</CardBody>
</Card>
```

## Props

- `Card`: 内部で`Stack`が使用されています。
	- `isFlex`: 横並びにできます。`<Stack>`ではなく`<Flex>`で構成されるようになります。
	- `href`:
	- `hover`
- `CardMedia`: `Frame`が内部で使用されています。
	- `aspect`: メディア比率
	- `w`: `isFlex`が指定されている場合、`<CardMedia>`の幅を指定できます。
- `CardBody`
	- `isLayer`?

radiusのデフォルト、`1`にする？

```js
import { Card, CardMedia, CardBody } from '@loos/lism-core';
```

#### 横並び時のメディアサイズについて

`<CardMedia>`に対して `w`で横幅、`aspect`でアスペクト比率を指定することでサイズを調整できます。（`w`の初期値は`50%`になっています）


<Alert preset="info">
横並び時、`.c--card__media` は `min-height:100%`になっており、コンテンツよりもメディアの縦幅が大きくなる場合のみ指定したアスペクト比が保持されます。
</Alert>

`"contentUp"`を指定しているので、 `.-hov:contentUp` が出力されます。
このクラスに対して、下記のCSSを追加して上記のDEMOを実装しています。

```scss
.c--card.-hov\:contentUp .c--card__hiddenArea{
	--gtr: 0fr;
	opacity: 0;
	transition: opacity .25s, grid-template-rows .25s;
}

.c--card.-hov\:contentUp:hover .c--card__hiddenArea{
	--gtr: 1fr;
	opacity: 1;
}
```
