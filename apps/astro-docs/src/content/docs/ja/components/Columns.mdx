指定した列数でカラムレイアウトを作成します。

<Preview>
	<PreviewTitle>対応するHTML構造</PreviewTitle>
	<PreviewCode>
	```jsx
	<div class="l--columns" style="--gtc:{cols}">...</div>
	```
	</PreviewCode>
</Preview>


## Props
- `cols`: 列数を指定します。ブレイクポイント指定可能です。
- その他、[`<Grid>`で使えるprops](/docs/components/Grid#props)が使えます。

## Import

```js
import { Columns } from '@loos/lism-core';
```
