[`is--container`](/ja/css/design-state/#is--container)をデフォルトで出力するモジュールです。

コンテナ要素としての役割をコード上でより明示的にわかりやすくするためにご利用ください。


## Props

- `size`: コンテナサイズを指定します。

## Import

```js
import { Container } from '@loos/lism-core';
```

## Usage

	<PreviewTitle>例</PreviewTitle>

### size の指定

通常のコンポーネントでは`isContainer='m'`のようにしてコンテナサイズを指定しますが、`Container`では`size='m'`のように書くことができます。

<Preview>
	<PreviewTitle>例</PreviewTitle>
	<PreviewArea resize p={20}>
		<Container size='xs' isFlow p='box' bgc='base-2'>
			<p>Content</p>
			<p>Content</p>
			<p>...</p>
		</Container>
	</PreviewArea>
	<PreviewCode>
		```jsx "size='xs'"
		<Container size='xs' isFlow p='box' bgc='base-2'>
			<p>Content</p>
			<p>Content</p>
			<p>...</p>
		</Container>
		```
	</PreviewCode>
</Preview>
