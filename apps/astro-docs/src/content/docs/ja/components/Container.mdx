[State Props](/docs/props/state-props) のうち、以下のPropsだけを限定して受け取れるようになっている特殊なコンポーネントです。  

- `isFlow`
- `isConstrained`
- `isContainer`
- `hasGutter`
- `isFullWide`
- `isWide`

さらに、isConstrained が指定されていない場合は自動で isContainer が指定されるようにもなっています。

コンテナ要素としての役割を明示的にしたい場合や、処理するPropsを限定させているので少しでも処理を軽くしたい場合に使用してください。

( [CSS Props](/docs/props/css-props) と as,tagなどは使用できます。 )
