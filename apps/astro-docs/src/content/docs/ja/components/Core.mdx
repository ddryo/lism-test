Lism Props( [CSS Props](/docs/props/css-props) ) を受け取るための特殊なコンポーネントです。  
他のライブラリで作成されたコンポーネントに対して Lism の CSS Props を使用したい場合などに利用できます。

[State Props](/docs/props/state-props) を受け取ることはできません。State Props を使いたい場合は、`<Layouter>`を使用してください。


何も指定がなければ、ただの`<div>`タグを返します。

	<PreviewTitle>対応するHTML構造</PreviewTitle>
	<PreviewCode>
		```jsx
		<div>...</div>
		```
	</PreviewCode>
</Preview>

## Props

- `as`: 展開コンポーネント
- `tag`: HTMLタグ

## Import

```js
import { Core } from '@loos/lism-core';
```
