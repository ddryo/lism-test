## Props

- `<Hero>`
- `<HeroContent>`
	- `type`

## Import

```js
import { Hero, HeroContent } from '@loos/lism-core';
```

### HeroContentの活用

HeroContent には`type`を指定でき、`top`、`center`、`bottom`のいずれかを指定することができます。  
それぞれのキーワードに合わせてpropsの初期値が変わります。

たとえば、`type='center'`の場合は、`py`, `my`, `hasGutter`, `isConstrained`が初期値に設定されます。

### Hero に高さをセットしたときに、HeroContent も高さを合わせる方法

`Hero` は `Stack` で構成されているので、`HeroContent`の`flex-grow`を`1`にすると、`Hero`の高さいっぱいまで伸びてくれます。

<Preview>
	<PreviewTitle>HeroContenの高さが広がる</PreviewTitle>
	<PreviewArea resize alignfull>
		<Hero minH='70vh' bd='guide' p="5" gap="5">
			<HeroContent type='top' p='em5' bd='guide'>
				<p>top</p>
			</HeroContent>
			<HeroContent fxg='1' bd='guide'>
				<Text fw='bold' fz='xl'>DEMO 7</Text>
				<DammyText length='l' />
			</HeroContent>
			<HeroContent type='bottom' p='em5' bd='guide'>
				<p>bottom</p>
			</HeroContent>
		</Hero>
	</PreviewArea>
	<PreviewCode alignfull>
		```jsx
		<Hero minH='70vh' bd='guide' p="5" gap="5">
			<HeroContent type='top' p='em5' bd='guide'>
				<p>top</p>
			</HeroContent>
			<HeroContent fxg='1' bd='guide'>
				<Text fw='bold' fz='xl'>DEMO 7</Text>
				<DammyText length='l' />
			</HeroContent>
			<HeroContent type='bottom' p='em5' bd='guide'>
				<p>bottom</p>
			</HeroContent>
		</Hero>
		```
	</PreviewCode>
</Preview>




ただし、これだけでは`HeroContent`直下の要素では `height` に`%`を使っても動作しません。

`HeroContent`直下の要素でもさらに`%`を使いたい場合は、次のような工夫が必要になります。

- `HeroContent`を`Grid`にして、`align-content:stretch`にする
- `HeroContent`内に`Layer`等を使ってコンテンツをabsolute配置する

前者の手法が簡単に扱えるように、`type="stretch"`を指定すると`as={Grid} ac='stretch' fxg='1'`をセットできるようにしています。


<Preview>
	<PreviewTitle>HeroContenをgrid化して、子要素でも`%`使えるようにする例</PreviewTitle>
	<PreviewArea resize alignfull>
		<Hero bd='guide' minH='70vh'>
			<MediaLayer src='/img.jpg' alt='' width={960} height={640}/>
			<FilterLayer bgc='black:20%' />
			<HeroContent type='stretch' hasGutter ai='center' bd='guide'>
				<Center minH='50%' isContainer='s' bg="glass" p="box" c="white" bdrs='s' gap={20}>
					<Text fw='bold' fz='xl'>DEMO 8</Text>
					<DammyText length='l' />
				</Center>
			</HeroContent>
		</Hero>
	</PreviewArea>
	<PreviewCode alignfull>
