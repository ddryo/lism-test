- Lismクラス: `.b--Note`

## 主なprops

- heading
- `isFlow`: `__body`に渡されます

<MemoBadge>Memo:</MemoBadge>
NoteHeader, NoteBody にわける？ → preset で色とアイコンの一括指定ができなくなる（ContextやChildren処理が必要）
## Import

```js
import { Note } from '@loos/lism-core';
```
