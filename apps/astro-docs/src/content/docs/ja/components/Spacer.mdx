## Props
- `w`: `width`の値を指定できます。
- `h`: `height`の値を指定できます。

<Alert preset="info">
`<Space>`では、`w`と`h`に対してSPACEトークンの値が使えます。
</Alert>


## Import

```js
import { Spacer } from '@loos/lism-core';
```
