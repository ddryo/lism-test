`bg`には、文字列またはオブジェクトを指定できます。

- 文字列指定の場合
	- ユーティリティクラス化できる文字列の場合は、`.-bg:{utility}`が出力されます。  
	(`stripe`, `grid`,`glass`)
	- それ以外の場合、`background`にそのまま出力されます。
- オブジェクト指定のときは、それぞれ対応するbackgroundプロパティとして出力されます。
	- bg: background
	- image → background-image
	- size → background-size,
	- repeat
	- position
	- color
	- attachment
	- blendMode
	- clip
	- origin
