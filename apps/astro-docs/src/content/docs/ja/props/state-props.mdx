[Design State](/ja/css/design-state) クラスに対応するLism Propsを紹介します。


<Alert preset='alert'>
ここで紹介するPropは、`<Layouter>`系コンポーネントでのみ有効です。(`<Core>`ではサポートされていません)
</Alert>


## isFlow

`.is--flow`に対応するPropです。
( <MemoBadge> 参考ページ </MemoBadge> : [.is--flow について](/docs/layout-state/is--flow) )


| Prop | Output |
|:--|:--|
| `isFlow` | `.is--flow` |
| `isFlow='s'` | `.is--flow:s` |
| `isFlow='l'` | `.is--flow:l` |
| `isFlow={value}` | `.is--flow:` & `--flowGap:{value}` |


<Preview>
	<PreviewCode slot='tab' label='コンポーネント'>
	```jsx
	<Layouter isFlow>...</Layouter>
	<Layouter isFlow="s">...</Layouter>
	<Layouter isFlow="20px">...</Layouter>
	```
	</PreviewCode>
	<PreviewCode slot='tab' label='出力'>
	```html
	<div class="is--flow">...</div>
	<div class="is--flow:s">...</div>
	<div class="is--flow:" style="--flowGap:20px">
	```
	</PreviewCode>
</Preview>

## isContainer

`.is--container`に対応するPropです。
( <MemoBadge> 参考ページ </MemoBadge> : [.is--container について](/docs/layout-state/content-size) )


| Prop | Output |
|:--|:--|
| `isContainer` | `.is--container` |
| `isContainer='s\|m\|l\|'` | `.is--container:s\|m\|l\|` |
| `isContainer={value}` | `.is--container:` & `--contentSize:{value}` |


<Preview>

</Preview>
