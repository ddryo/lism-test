# Alert

短い文言を目立たせて表示したい時に使えるコンポーネントです。

- `<Alert>`
- Lismクラス: `b--alert`

- デフォルトkeycolor=main


## 主なProps
- `preset`: `"memo"` | `"error"` ...
- `color`
- `icon`
- `iconSize`
{/* - `variant`: `"fill"` | `"outline"` ? */}


## Import

```js
import { Alert } from '@lism/core';
```
