# Badge

バッジを表示するコンポーネントです。

<Demo>
	<Demo.Title>対応するHTML構造</Demo.Title>
	<Demo.Code>
		```jsx
		<span class="c--badge">Badge</span>
		```
	</Demo.Code>
</Demo>


## 主なprops
- `keycolor`: バッジのキーカラー。デフォルトは`main`カラーです。
	- パレットカラーを指定できます。(`"red"`, `"blue"`, `"green"`, `"yellow"`, `"purple"`, `"orange"`, `"pink"`, `"gray"`
- `variant`: ボタンの表示スタイル。`outline`、`subtle`の2種類を標準で用意しています。
	- `c--badge--{variant}`がクラスとして出力されます。

## Import

```js
import { Badge } from '@lism/core';
```

## DEMO

export const ColorDemos = ({colors, variant}) => {
	return (
		<Cluster gap={20}>
			{colors.map((ckey) => {
				return (
					<Badge keycolor={ckey} key={ckey} variant={variant || ''}>"{ckey}"</Badge>
				);
			})}

		</Cluster>
	);
}

### アイコン

`display:inline-flex`, `align-item:center` を `.c--badge` に対して指定していますので、内部にアイコンを配置してもきれいに表示されます。

	<Demo.Title>`<InlineIcon>`の使用例</Demo.Title>
	<Demo.Preview resize p={20}>

			<Badge><Icon icon={Book} size="1.25em"/> Badge</Badge>
			<Badge variant="outline"><Icon icon={Cat} size="1.25em"/> Badge</Badge>
		</Cluster>
	</Demo.Preview>
<Demo.Code>
```jsx
<Cluster gap={20}>
	<Badge>
		<Icon icon={Book} size="1.25em"/> Badge
	</Badge>

		<Icon icon={Cat} size="1.25em"/> Badge
	</Badge>
</Cluster>
```
