# Box

コンテンツをラップしてグループ化するためのシンプルなコンポーネントです。

<Demo>
	<Demo.Title>対応するHTML構造</Demo.Title>
	<Demo.Code>
	```jsx
	<div class="l--box">...</div>
	```
	</Demo.Code>
</Demo>

## Import

```js
import { Box } from '@loos/lism-core';
```
