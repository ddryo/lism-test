# Button

- `<Button>`
- Lismクラス: `.l--button` → `.e--button` にすべき?

## 主なprops
- `color`: ボタンのキーカラー。デフォルトは`main`カラーです。
- `variant`: ボタンの表示スタイル。`fill`(default)、`outline`、`ghost`の3種類。
- `icon`: アイコン。`icon`に指定するとアイコンのみ表示される。

{/* ## 注意点
Common Props の `w`, `h` ではなく、`width`, `height` を使ってください。 */}

## hover="reverse"

Button用のhoverユーティリティ。

- 基本的には、 cとbgcを入れ替えるようになっている
- fill で使うと、文字色,ボーダー色→ホバー前の背景色になり、背景→透明
- outline,ghost で使うと、文字色→ filled--c, 背景色→ホバー前の文字色
	- テキストカラー反転してるようなエリアでは、ホバー時の文字色を明示的に指定してあげないと真っ白になったり真っ黒になったりする。
