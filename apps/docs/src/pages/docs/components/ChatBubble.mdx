# ChatBubble

- `<ChatBubble>`
- Lismクラス: `.b--chatBubble`

## 検討

- アイコン側、吹き出し側でコンポーネントわける？
- ふきだしテキスト
- コンポーネントにする必要はなく、WPテーマにだけ搭載する？


## Import

```js
import { ChatBubble } from '@lism/core';
```
