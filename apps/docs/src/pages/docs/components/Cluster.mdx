import { Cluster, Badge, Item, Lism } from '@lism/core';

# Cluster

複数のちさな要素をFlexレイアウトで**横並び**に配置するためのコンポーネントです。

`flex-wrap:wrap`がデフォルトで適用されている[`<Flex>`](/docs/components/Flex)ベースのコンポーネントで、`<Flex wrap="wrap">`と機能的には同等です。

<Demo>
	<Demo.Title>対応するHTML構造</Demo.Title>
	<Demo.Code>
	```jsx
	<div class="l--cluster">...</div>
	```
	</Demo.Code>
</Demo>


## Props

- `itemMinW`: 子要素の最低限維持する幅
	- ※ `.is--item, span, li, a, button`に対してのみ有効です。
- その他、[`<Flex>`で使えるprops](/docs/components/Flex#props)が使えます。

`gap`に初期値`20`が指定されています。


## Import

```js
import { Cluster } from '@lism/core';
```
