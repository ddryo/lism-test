import { Cluster, Badge } from '@lism/core';

# Cluster

複数の要素を**Flexレイアウトで横並びに配置する**ためのコンポーネントです。

`--fxw:wrap`がデフォルトで適用されている[`isFlex`](/docs/layout-system/is-flex)なコンポーネントで、`<Flex fxw="wrap">`と機能的には同等です。

- `<Cluster>`
- Lismクラス: `.l--cluster.is--flex`
- itemMinW: 子要素の最低限維持する幅
	- ※ `div, span, li, p, a, button, input`に対してのみ有効です。

## Import

```js
import { Cluster } from '@lism/core';
```
