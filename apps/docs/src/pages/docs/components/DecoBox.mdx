# DecoBox

コンテンツを装飾するためのボックスモジュール。

対応クラスは`.c--decoBox--{variant}`。  
(基本的にどういう装飾を行うかの指定が必須で`.c--decoBox`に何かスタイルがあるわけではないので、modifier付きクラス単体で使う）


## Props

- variant
- subvariant
- data
	- decoratorCount
	- boxProps
	- bodyProps
	- decoratorProps
- isFlow, flowGap は body側に渡されます.


variant, subvariant に対しての 各Props をプリセット登録できるようにする予定。プリセット登録せずに装飾内容を指定する場合は `data` に直接指定する。

## Import

```js
import { DecoBox } from '@lism/core';
```
