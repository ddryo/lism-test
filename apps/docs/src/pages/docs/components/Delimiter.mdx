import { Cluster, Delimiter } from '@loos/lism-core';

# Delimiter

テキストレベルの小さな要素を区切るための要素。

<Demo>
	<Demo.Title>対応するHTML構造</Demo.Title>
	<Demo.Code>
	```jsx
	<div class="e--delimiter">...</div>
	```
	</Demo.Code>
</Demo>

デフォルトで等幅フォントが適用されるため、記号でデータを区切る際に便利になっています。

## Props

--


## Import

```js
import { Delimiter } from '@loos/lism-core';
```
