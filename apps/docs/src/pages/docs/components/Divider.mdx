# Divider

境界線

- `<Divider>`
- Lismクラス: `.l--divider`

## Props
- `level`: 高さのレベル。正の数・負の数を指定できます。 `5` がデフォルトです。
	- `{level} * 1cqw` で`<Divider>`の高さが計算され、600px時点の値が最小値、1200px時点の値が最大値となります。
- `stretch`: 横方向の伸縮率。 `1` がデフォルトです。
- `offset`: X方向の位置をずらす。 `stretch`と合わせて`%`単位で指定してください。
	- 例: `stretch:1.5` なら、`offset:-25%~25%`の範囲が有効です。
- `flip`: x軸、y軸方向の反転。 `x` `y` `xy` が指定できます。
