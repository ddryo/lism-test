# FilterLayer

`<Layer>`の拡張モジュール。  
親サイズにfitするサイズで配置され、bgc, blur, contrast, brightness を指定できる。
`<MediaLayer>` と組み合わせると便利。

- `<FilterLayer>`
- Lismクラス: `.l--layer--filter`


## 主なProps
- `blur` : ブラー

## Import

```js
import { FilterLayer } from '@lism/core';
```
