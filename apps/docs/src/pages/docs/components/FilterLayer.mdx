# FilterLayer

**背景にエフェクト効果を加える**ための、[`<Layer>`](/docs/components/Layer)の拡張モジュール。

`backdrop-filter`の主要なプロパティをオブジェクトで指定できるようになっています。

<Demo>
	<Demo.Title>対応するHTML構造</Demo.Title>
	<Demo.Code>
	```jsx
	<div class="l--layer l--layer--filter">...</div>
	```
	</Demo.Code>
</Demo>


## Props
- `backdropFilter`の個別指定
	- `blur`,
	- `contrast`,
	- `brightness`,
	- `grayscale`,
	- `hue-rotate`,
	- `invert`,
	- `saturate`,
	- `sepia`,
- `backdropFilter` : `css.backdropFilter`に直接値を渡します。filterトークンを指定する時にはこちらを使います。
- texture?


<Alert preset="info">
backdropFilterの指定がある時、blur, brightnessなどの個別指定は無視されます。
</Alert>

## Import

```js
import { FilterLayer } from '@loos/lism-core';
```
