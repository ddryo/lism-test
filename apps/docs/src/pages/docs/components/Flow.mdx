import { Box, Flow } from '@loos/lism-core';

# Flow

[`.is--flow`](http://localhost:3000/docs/props/is-flow)を出力するコンポーネントで、 `<Lism isFlow>`のエイリアスです。


<Demo>
	<Demo.Title>対応するHTML構造</Demo.Title>
	<Demo.Code>
	```jsx
	<div class="is--flow">...</div>
	```
	</Demo.Code>
</Demo>


## Props

- `gap`: `Flow`コンポーネントでのみ、`gap` = `flowGap`として扱われます。
- `as`: 展開したいコンポーネントを指定できます。
	- `<Flow as="Box">` = `<Box isFlow>` です。

## Import

```js
import { Flow } from '@loos/lism-core';
```
