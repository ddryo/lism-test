# FluidFix

片方の横幅を固定し、もう片方の横幅が伸縮する横並びレイアウトで、かつ、指定した値を下回ると自動で縦並びに切り替わるレイアウトを組めるモジュールです。

<Alert preset="point">
コンテナクエリに依存せずに横並びと縦並びの切り替えを行います。
</Alert>

- `<FluidFix>`
- Lismクラス: `.l--fluidFix`


## 主なProps

- `fixW` : 固定幅
	- デフォルト値: `auto` (要検討)
- `fluidMinW` : Fluid要素の最小幅
	- デフォルト値: `50%`
- `fix`: `first` or `last` で FIX要素を指定。
	- `data-fix`属性に指定した文字列が出力されます。
{/* - `gap`: 初期値 `20` */}



## Import

```js
import { FluidFix } from '@lism/core';
```
