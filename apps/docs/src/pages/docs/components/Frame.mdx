# Frame

横縦比率(`aspect-ratio`)を指定したり、高さを指定することで、内部のコンテンツをそのサイズからはみ出さないように表示できるモジュール。

<Demo>
	<Demo.Title>対応するHTML構造</Demo.Title>
	<Demo.Code>
	```jsx
	<div class="l--frame -aspect:{aspect}">...</div>
	```
	</Demo.Code>
</Demo>



## props
- `aspect` : アスペクト比率を指定する。ブレイクポイント指定可能です。
	- 例: `aspect="16/9"`, `aspect={["4/3", "16/9"]}`

{/* <Alert preset="info">
	一応、`aspect` は 単純に Prop としても使用できます。
</Alert> */}


## Import

```js
import { Frame } from '@loos/lism-core';
```

### 数値以外での比率指定

- `golden`: 黄金比
- `silver`: 白銀比
- `bronze`: 青銅比
- `ogp`: OG Imageの比率

export const FrameRatios = () => {
	return (
		<Columns cols={[1, 2, 4]} gap={20} >
			{["golden", "silver", "bronze", "ogp"].map((ratio) => {
				return (
					<Frame key={ratio} aspect={ratio} bgc="whitesmoke">
						<Layer position="center" c="gray" fz="s">
							<code>aspect="{ratio}"</code>
						</Layer>
					</Frame>
				);
			})}
		</Columns>
	);
}
