# Grid

コンテンツを**Gridレイアウトで配置する**ためのコンポーネントです。

`<Grid>` は `<Box>`に[`isGrid`](/docs/layout-system/is-grid)を指定した状態のコンポーネントで、`<Box isGrid>`と同等です。

-   `<Grid>`
-   Lismクラス: `.l--box.is--grid`

用途に合わせて Columns, RatioGrid, TileGrid などを用意していますが、内部ではこのGridを使用している。

`<Grid>`は、それらより柔軟に各プロパティを指定して利用できる。


## Import

```js
import { Grid } from '@lism/core';
```
