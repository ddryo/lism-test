# Icon

- `<Icon>`
	- Lismクラス: `.e--icon`
- `<InlineIcon>`: テキストの中にアイコンをインライン要素として挿入するためのコンポーネント。
	- Lismクラス: `.e--inlineIcon`


## 主なprops
- `icon`: アイコンの指定。svgコンポーネントを投げるか、文字列で指定するとCSSでアイコンを定義できるようにdata-css-iconとして出力される。
- `size`: svgタグのwidth,heightに出力されるサイズ。CSSアイコンの場合は--sizeとして出力される。
	- default: `1em`


## 補足事項
- fillがcurrentColorになっているアイコンのみ、色を変えられる。


## Import

```js
import { Icon } from '@lism/core';
```

InlineIcon（e--icon）に対しては次のようなCSSが適用されており、文字と並んでも見やすく鳴るような調整がデフォルトで入っています。

```css
.e--inlineIcon {
    display: inline;
    vertical-align: -0.125em;
    margin-inline: 0.125em;
    transform: scale(1.125);
}
```
