# Layout

Lism Props( [CSS Props](/docs/props/css-props) 及び [State Props](/docs/props/state-props) )を受け取るための特殊なコンポーネントです。  
他のライブラリで作成されたコンポーネントに対して Lism Props を使用したい場合などに利用できます。

何も指定がなければ、ただの`<div>`タグを返します。

<Demo>
	<Demo.Title>対応するHTML構造</Demo.Title>
	<Demo.Code>
		```jsx
		<div>...</div>
		```
	</Demo.Code>
</Demo>

## Props
- `as`: 展開コンポーネント
- `tag`: HTMLタグ

## Import

```js
import { Layout } from '@loos/lism-core';
```
