# Lism


<Alert>
廃止予定. → `<Layout>`, `<Core>`に差し替え
</Alert>

Lism Props を受け取るための特殊なコンポーネントです。他のライブラリで作成されたコンポーネントに対して Lism Props を使用したい場合などに利用できます。

何も指定がなければ、ただの`<div>`タグを返します。

<Demo>
	<Demo.Title>対応するHTML構造</Demo.Title>
	<Demo.Code>
		```jsx
		<div>...</div>
		```
	</Demo.Code>
</Demo>

## Props
- `as`: 展開コンポーネント

## Import

```js
import { Lism } from '@loos/lism-core';
```
