# MediaLayer

`<Layer>`の拡張モジュール。  
親サイズにフィットするサイズで`object-fit` なメディア（画像・動画）を配置するレイヤー。

- `<MediaLayer>`
- Lismクラス: `.l--layer--media`
- 直下のimg要素には`.l--layer--media__img`が付与される.


## 主なProps
- `tag`
- `z`: ラッパー要素である `<Layer>`に渡されます。
- `...props` : メディアデータ指定. `src`, `alt` `width`, `height` など

<Alert>
`media`を指定せず、 `<MediaLayer><img ... /></MediaLayer>` のように`children`にそのまま画像を渡してもOKです。
</Alert>


## Import

```js
import { MediaLayer } from '@lism/core';
```
