# RatioGrid

子要素の幅の比率を`1:2`や`1:1:2`で指定して並べることができるグリッドレイアウト。

- `<RatioGrid>`
- Lismクラス: `.l--ratioGrid`

## 主なProps

- `ratio`: ベースの比率


## Import

```js
import { RatioGrid } from '@lism/core';
```
