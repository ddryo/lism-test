# RatioGrid

子要素のサイズ比率（`1:2`や`1:1:2`など）を指定して配置できるグリッドモジュールです。

<Demo>
	<Demo.Title>対応するHTML構造</Demo.Title>
	<Demo.Code>
	```jsx
	<div class="l--ratioGrid" style="--gtc:{ratio}">...</div>
	```
	</Demo.Code>
</Demo>


## Props

- `ratio`: 要素のサイズ比率を指定します。ブレイクポイント指定可能です。
- その他、[`<Grid>`で使えるprops](/docs/components/Grid#props)が使えます。


## Import

```js
import { RatioGrid } from '@lism/core';
```
