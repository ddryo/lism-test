# Reel


<Reel>
	<ReelContents />
</Reel>

<Reel unreel='md' itemW='50%'>
	<ReelContents />
</Reel>


コンテンツを横スクロールで表示するコンポーネントです。

- `<Reel>`
- Lismクラス: `.l--reel` → `is--reel`にする?


## 主なProps

- `itemW`: 各アイテムの横幅（クエリ対応すべき？）
- `componentForWide` ?:  wide幅の時のコンポーネントを指定する。SPサイズの時のみReelにするための設定

## 注意点

`<Reel>` 直下の要素には `<Item>` を使ってください。


### 要検討

- スクロールバーの色調整機能
- align-itemsをストレッチ以外にできるようにするか？
- l--box--reel にする？
- スクロール可能であることを視覚的に表示する機能をつける？
	- JSを使う?
	- lea verou 形式のCSSで端にシャドウつける？（背景色と一致させないといけない）

- ブロック化した時
	- 子要素はBox or 画像・動画だけにする？
	- もしくは、専用の子ブロック（BOX拡張）のみにするか


## Import

```js
import { Reel } from '@lism/core';
```
