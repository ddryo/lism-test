# SideFlex


**要素の横幅を固定してサイドに配置し、もう一方の要素幅が伸縮する**ような２カラムレイアウトで、かつ、**指定した横幅を下回ると自動で縦並びに切り替わる**を組めるモジュールです。

<Alert preset="point">
コンテナクエリに依存せずに横並びと縦並びの切り替えを行います。
</Alert>


## Props

| プロパティ | 説明 | デフォルト値 |
|---|---|---|
| `sideW` <PropBadge type="cssvar">--side--w</PropBadge> | Side要素の横幅 | `auto` |
| `mainMinW` <PropBadge type="cssvar">--main--minW</PropBadge>| もう一方の要素が維持する最小幅 |`50%`|

その他、[`<Flex>`で使えるprops](/docs/components/Flex#props)が使えます。


## Import

```js
import { SideFlex } from '@loos/lism-core';
```

### 折り返さずに使う例

`mainMinW`を小さな値にすることで小さいデバイスでも折り返さないことも可能です。

			<FlexItem bd="guide">
				<DammyText />
			</FlexItem>
			<FlexItem isSide bd w={['12em', '16em']}>Side Content</FlexItem>
		</SideFlex>
	</Demo.Preview>
	<Demo.Code>

		```jsx
			<FlexItem>

		</SideFlex>
		```
	</Demo.Code>
</Demo>
