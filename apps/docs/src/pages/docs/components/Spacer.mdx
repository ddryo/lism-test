import { Spacer } from '@lism/core';


# Spacer

- `<Spacer>`
- Lismクラス: `.l--spacer`

## 主なprops
- width: 横幅を指定できます。
- height: 高さを指定できます。

スペーシングプリセットがサイズ指定に使えます。

## 注意点
Common Props の `w`, `h` ではなく、`width`, `height` を使ってください。


## Import

```js
import { Spacer } from '@lism/core';
```
