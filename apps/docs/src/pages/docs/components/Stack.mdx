import { Stack } from '@lism/core';

# Stack

複数の要素をFlexレイアウトで**縦並び**に配置するためのコンポーネントです。

`flex-direction:column`がデフォルトで適用されている[`<Flex>`](/docs/components/Flex)ベースのコンポーネントで、`<Flex direction="column">`と機能的には同等です。

<Demo>
	<Demo.Title>対応するHTML構造</Demo.Title>
	<Demo.Code>
	```jsx
	<div class="l--stack">...</div>
	```
	</Demo.Code>
</Demo>



## Props

[`<Flex>`で使えるprops](/docs/components/Flex#props)が使えます。



## Import

```js
import { Stack } from '@lism/core';
```

## Stackを活用したセンター寄せレイアウト

`<Stack ai="center">`( `.l--stack.-ai:c` )の組み合わせで、Every Layout で言うところの **"内在的な中央寄せ"** （コンテンツが短い時だけ中央寄せ）にできます。

<Demo className="has--box-guide">
	<Demo.Title>内在的な中央寄せ</Demo.Title>
	<Demo.Preview resize>
		<Stack ai="center" gap={30} p={40} lismState={["has--itemGuide"]}>
			<h3 role="presentation">
				見出し
			</h3>
			<DammyText length="xs" />
			<DammyText length="l" />
		</Stack>
	</Demo.Preview>
	<Demo.Code>
	```jsx
	<Stack ai="center" gap={30} p={40}>
		<h3>見出し</h3>
		<p>...</p>
		<p>...</p>
	</Stack>
	```
	</Demo.Code>
</Demo>
