## Props
- TermList
	- mode : 'stack'(default) || 'fluid',
	- fixW : 'fluid'モード時の、dtの固定幅
	- fluidMinW : 'fluid'モード時の、ddの最小幅

- TermListRow
	- ( mode )
		- ( fixW )
		- ( fluidMinW )
	- term : ターム名。(dt要素の内容)
	- dtProps : dtに渡したいProps
	- ddProps : ddに渡したいProps
	-  ifFlow, flowGap : dd 側に渡るようになっています。


## Import

```js
import { TermList, TermListRow } from '@loos/lism-core';
```

### `mode="fluid"`時の挙動

`mode="fluid"`を指定すると、`<TermListRow>`が`<FluidFix>`で構成されるようになります。
`<FluidFix>`用の`fixW`, `fluidMinW` は、親である`<TermList>`に指定すれば子に引き継がれるようになっています。

デフォルト値はそれぞれ次の通りです。

- `fixW`: `10rem`
- `fluidMinW`: `15rem`

<Alert>
`mode="fluid"`を親である`<TermList>`に指定できるのはReactベースのフレームワークのみ。`React.Children.map`を使って`<TermListRow>`に渡しているので、Astroではエラーになるはず...。（要検証）
</Alert>

↓ Astro等では次のように各`<TermListRow>`に`mode`を指定し、`<TermList>`に`fixW`,`fluidMinW`の明示的な指定が必要です。
