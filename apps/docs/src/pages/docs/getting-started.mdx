## Import Components

特定の環境に合わせた コンポーネントの読み込み

<Tabs>
	<TabItem label='React'>
		```bash
		npm i @loos/lism-core
		```

		```js
		import { Box, ... } from '@loos/lism-core';
		```
	</TabItem>
	<TabItem label='Next.js'>
		```bash
		npm i @loos/lism-core @loos/lism-next
		```

		```js
		import { Box, ... } from '@loos/lism-next';
		```
	</TabItem>

	<TabItem label='Astro'>
		```bash
		npm i @loos/lism-core @loos/lism-astro
		```

		```js
		import { Box, ... } from '@loos/lism-astro';
		```
	</TabItem>
</Tabs>


### Use Components

```jsx
import { Box, Cluster, ... } from '@loos/lism-core';

<Box p={20}>
	<Cluster>...</Cluster>
	...
</Box>
```
