# Installation

<Alert>
今はまだパッケージの公開をしていません。[github](https://github.com/ddryo/lism-test)からソースコードをダウンロードしてビルド後、そのソース内の`apps/vite-playground`などで利用できます。
</Alert>
<Alert>
しかも`@lism`がすでにnpm上で使われていたので、譲渡申請が通らなければ変更予定。
</Alert>

<Tabs>
	<TabItem label='npm'>
		```bash
		npm i @lism/core
		```
	</TabItem>
	<TabItem label='pnpm'>
		```bash
		pnpm add @lism/core
		```
	</TabItem>
		<TabItem label='yarn'>
		```bash
		yarn add @lism/core
		```
	</TabItem>
</Tabs>

## Import Components

特定の環境に合わせた コンポーネントの読み込み

<Tabs>
	<TabItem label='React'>
		```bash
		npm i @lism/core
		```

		```js
		import { Box, ... } from '@lism/core';
		```
	</TabItem>
	<TabItem label='Next.js'>
		```bash
		npm i @lism/core @lism/next
		```

		```js
		import { Box, ... } from '@lism/next';
		```
	</TabItem>

	<TabItem label='Astro'>
		```bash
		npm i @lism/core @lism/astro
		```

		```js
		import { Box, ... } from '@lism/astro';
		```
	</TabItem>
</Tabs>
