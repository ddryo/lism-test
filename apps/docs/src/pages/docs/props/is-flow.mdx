# isFlow

Lism設計では、**コンテンツの間隔を`margin-block-start`で管理するためのStateクラス**として、`.is--flow`を用意しています。(<MemoBadge> 参考ページ </MemoBadge> : [コンテンツ間隔の制御クラス .is--flow について](/docs/design-system/contents-layout#is--flow))


そのステートクラスに対応する Lism Props が、`isFlow`です。  
Lismコンポーネントに`isFlow`を指定することで、`.is--flow`クラスが出力されるようになっています。

<Alert preset='info'>
`isFlow` は `<Layout>`及び`<Layout>`で構成されるコンポーネントでのみ有効です。
</Alert>

それ以外のケースでどうしても`isFlow`を使いたい場合は、直接 `className='is--flow'` を指定してください。

	</Demo.Code>
</Demo>

### Flowレイアウト直下の余白量を一括で変更する方法

`isFlow`に値を指定すると、flowレイアウトの余白を調整できます。  
指定する値が専用のトークン値（`s`, `m`, `l`）の時、`.is--flow:s` のようにクラスが変化します。

<Note preset="info" heading="注意ポイント">
- `isFlow`はブレークポイント指定には対応していません。
</Note>


<Demo className="has--box-guide">
	<Demo.Title>`flowGap`を `s` にしてみる</Demo.Title>
	<Demo.Preview resize p={30}>
		<Layout isFlow="s">
			<p className='-bd:guide'>コンテンツ</p>
			<p className='-bd:guide'>コンテンツ</p>
			<p className='-bd:guide'>コンテンツ</p>
		</Layout>
	</Demo.Preview>
	<Demo.Code>
	```jsx
	<Layout isFlow="s">
		<p>...</p>
		<p>...</p>
		<p>...</p>
	</Layout>
	```
	</Demo.Code>
</Demo>
