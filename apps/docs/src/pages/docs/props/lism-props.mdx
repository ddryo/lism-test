## Margin and padding



|Prop|CSS Property|Token,Utility|
|---|---|---|
|p|padding|SPACE, PADDING|
|pl|padding-left|SPACE, PADDING|
|pr|padding-right|SPACE, PADDING|
|pt|padding-top|SPACE, PADDING|
|pb|padding-bottom|SPACE, PADDING|
|px|padding-left, padding-right|SPACE, PADDING|
|py|padding-top, padding-bottom|SPACE, PADDING|
|ps|padding-start|SPACE, PADDING|
|pbs|padding-block-start|SPACE, PADDING|
|m|margin|SPACE|
|ml|margin-left|SPACE|
|mr|margin-right|SPACE|
|mt|margin-top|SPACE|
|mb|margin-bottom|SPACE|
|mx|margin-left, margin-right|SPACE|
|my|margin-top, margin-bottom|SPACE|
|ms|margin-start|SPACE|
|mbs|margin-block-start|SPACE|

<Reference>
[SPACE Tokens](/docs/tokens/spacing)
</Reference>

Utility Class: 
- paddings
	- `-p[X|Y]:{0,5,10,15,20,30,40,50}`
	- `-p:{box[:s|:l]}`
- margins
	- `-mbs:{0,5,10,15,20,30,40,50}`, s,m,l?
	- `-m[l|r|t|b|y|x]:auto`

全てBP対応?

<Note>
論理プロパティは ps, pbs だけでいいかどうかは要検討.

- inline-start, block-start だけでいいか。end方向はX,Yとstartの併用で擬似的に指定できる。
- px,pyをinline,blockとしてもいいかもしれない。
</Note>

<Demo>
	<Demo.Title>Examples</Demo.Title>
	<Demo.Code>
	```jsx
	<Box m={40}>...</Box>
	<Box pb='20px'>...</Box>
	<Box mx='auto'>...</Box>
	<Box p='box' py={[null, 50]}>...</Box>
	<Text px={[20, 30]}>...</Text>
	```
	</Demo.Code>
</Demo>

<Demo>
	<Demo.Title>各方向の値をp,mに対してオブジェクトで指定することも可能</Demo.Title>
	<Demo.Code>
	```jsx
	<Box m={{left:20, bottom: 30}}>...</Box>
	<Box p={{x:30, y: 40}}>...</Box>
	```
	</Demo.Code>
</Demo>

<Alert preset="info">
オブジェクト指定は x,y,top,right,bottom,left にのみ対応しています。 
</Alert>



## gap

Flex, Gridでgapを指定できます。

|Prop|CSS Property|Token,Utility|
|---|---|---|
|gap|gap|SPACE|
|rowGap|row-gap|SPACE|
|columnGap|column-gap|SPACE|

<Reference>
[SPACE Tokens](/docs/tokens/spacing)
</Reference>

Utility Class: 
- `-gap:{0,5,10,15,20,30,40,50}`
- em系？

全てBP対応
