# State Props

LayoutStateクラスに対応するPropsを紹介します。


<Alert preset='info'>
ここで紹介するPropは、`<Layouter>`系コンポーネントでのみ有効です。
</Alert>


## isFlow

`.is--flow`に対応するPropです。
( <MemoBadge> 参考ページ </MemoBadge> : [.is--flow について](/docs/layout-state/is--flow) )


| Prop | Output |
|:--|:--|
| `isFlow` | `.is--flow` |
| `isFlow='s'` | `.is--flow:s` |
| `isFlow='l'` | `.is--flow:l` |
| `isFlow={value}` | `.is--flow:` & `--flowGap:{value}` |


<Demo>
	<Demo.Title>`isFlow`</Demo.Title>
	<Demo.Preview resize p={5}>
		<Layouter isFlow>
			<p className='-bd:guide'>コンテンツ</p>
			<p className='-bd:guide'>コンテンツ</p>
			<p className='-bd:guide'>コンテンツ</p>
		</Layouter>
	</Demo.Preview>
	<Demo.Code>
	```jsx
	<Layouter isFlow>
		<p>...</p>
		<p>...</p>
		<p>...</p>
	</Layouter>
	```
	</Demo.Code>
</Demo>

<Demo>
	<Demo.Title>その他の例</Demo.Title>
	<Demo.Code>
	```jsx
	<Box isFlow='s'>...</Box>
	<Box isFlow={30}>...</Box>
	<Box isFlow='10px'>...</Box>
	```
	</Demo.Code>
</Demo>
