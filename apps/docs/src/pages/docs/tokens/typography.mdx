### スケーリングについて

各プリセット値は、数学的に美しいとされるスケーリング理論に基づいて定義することを推奨しています。

<Alert>
デフォルトでは、**分子8の調和数列**ベースで設計しています。
</Alert>

<Alert>
	Memo: 英語: `8`ベース、 日本語: `9`ベース でスケーリングするのが良さそう。
</Alert>

参考: [音楽、数学、タイポグラフィ](https://standard.shiftbrain.com/blog/music-math-typography)

## Customize

以下が初期値です。
スタイルシートにコピーして別の値へカスタマイズしてください。

```css filename="your css"
@layer lism-init {
	:root{
		
		/* 400px:0.875rem ~ 600:1rem */
		/* --fz--root: clamp(0.875rem, 0.625rem + 1vw, 1rem); */

		/* 350px:0.875rem ~ 600:1rem */
		--fz--root: clamp(0.875rem, 0.7rem + 0.8vw, 1rem);
		--lh--bsae: var(--lh--70);
	}

	h1 {
		--fz: var(--fz--3xl);
		--lh: var(--lh--40);
	}
	h2 {
		--fz: var(--fz--2xl);
		--lh: var(--lh--50);
	}
	h3 {
		--fz: var(--fz--xl);
		--lh: var(--lh--50);
	}
	h4 {
		--fz: var(--fz--l);
		--lh: var(--lh--60);
	}
	h5 {
		--fz: var(--fz--m);
	}
	h6 {
		--fz: var(--fz--s);
		--lh: var(--lh--60);
	}
}

body{
    font-size: 1rem;
    line-height: var(--lh--bsae);
}

```
