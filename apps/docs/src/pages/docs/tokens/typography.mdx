### スケーリングについて

各プリセット値は、数学的に美しいとされるスケーリング理論に基づいて定義することを推奨しています。

<Alert preset="info">
デフォルトでは、**分子8の調和数列**ベースで設計しています。
</Alert>

<Alert preset="info">
	Memo: 英語: `8`ベース、 日本語: `9`ベース でスケーリングするのが良さそう。
</Alert>

参考: [音楽、数学、タイポグラフィ](https://standard.shiftbrain.com/blog/music-math-typography)
