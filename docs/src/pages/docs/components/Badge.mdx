# Badge

- `<Badge>`
- Lismクラス: `.e--badge`


## 主なprops
- `color`: バッジのキーカラー。デフォルトは`main`カラーです。
	- パレットカラーを指定できます。(`"red"`, `"blue"`, `"green"`, `"yellow"`, `"purple"`, `"orange"`, `"pink"`, `"gray"`
- `variant`: ボタンの表示スタイル。`fill`(default)、`outline`、`text`の3種類。

## Import


```js
import { Badge } from '@lism/core';
```

## Demo

export const ColorDemos = ({variant = 'fill'}) => {
	return (
		<Cluster gap={20}>
			{["red", "orange", "yellow", "green", "blue", "purple", "pink", "gray", "main", "accent"].map((ckey) => {
				return (
					<Badge color={ckey} key={ckey} variant={variant}>Badge</Badge>
				);
			})}

		</Cluster>
	);
}
