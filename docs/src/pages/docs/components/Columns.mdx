import { Columns } from '@lism/core';

# Columns

- `<Columns>`
- Lismクラス: `.l--columns`
- grid で実装。(flexだと gap の値を計算に含める必要がでてくるが、grid だと 1fr で自動調整してくれる。)
{/* - default gap: 40 ? */}

## 主なProps
- `cols`: 列数指定


## Import

```js
import { Columns } from '@lism/core';
```
