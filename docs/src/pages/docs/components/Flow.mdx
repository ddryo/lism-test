import { Box, Flow } from '@lism/core';

# Flow

[`.is--flow`](http://localhost:3000/docs/props/is-flow)を出力するコンポーネントです。

- `<Flow>`
- Lismクラス: `.is--flow`

`<Lism isFlow>`のエイリアス。


## Props

- `gap`: `Flow`コンポーネントでのみ、`gap` = `flowGap`として扱われます。
- `as`: 展開コンポーネントを指定できます。
	- `<Flow as="Box">` = `<Box isFlow>` です。

## Import

```js
import { Flow } from '@lism/core';
```
