# Hover

- `hover`にhover時の挙動を指定できる。
- `hover="_utilName_"`でユーティリティクラスを出力。
	- `.-hov:_utilName_`が出力される。
	- `"fade"`, `"dark"`, `"shadow"` などが指定可能。
- `hover={["_utilName1_", "_utilName2_"]}`で複数のユーティリティクラスを出力。
- `hover={{c,bgc,bdc,radius,shadow,utility}}` でhover時のpropを詳細に指定可能。
	- c: color
	- bgc: background-color
	- bdc: border-color
	- radius: border-radius
	- shadow: box-shadow
	- utility: ユーティリティクラス用のキーワード値 ( `"fade"`, `"dark"`, `"shadow"` など )

<h2>カスタムUtility</h2>

予め用意されているユーティリティ用のプリセット値以外を指定した時も、その文字列がユーティリティクラス名として出力されます。
例えば、`hover="hoge"`とすると`.-hov:hoge`が出力されます。

これにより、自身でユーティリティクラスを作成してより自由にhover効果を適用可能です。


```jsx
<LinkBox hover="hoge">
	...
</LinkBox>
```

↓ CSS

```css
.-hov\:hoge:hover {
	/* ...自由にスタイルを... */
}
```
