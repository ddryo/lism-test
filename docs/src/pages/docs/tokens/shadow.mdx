import { Box,Frame, Text, Columns, Flex, Stack } from '@lism/core';

# Shadow

## Tokens
- Tokens: `-4`, `-3`, ..., `0`, ... `3`, `4`

## Utility
- Utility Class: `-bxsh:-4`, ... ,`-bxsh:4`


## Variables

- `--shadow--{token}`
	- プラス値: `--shadow--1`, `--shadow--2`...
	- マイナス値: `--shadow--i1`, `--shadow--i2`...
- 構成変数
	- `--shadow--hsl`: `0 0% 0%`;
	- `--shadow-strength--base`: `0.05`
	- `--shadow-strength`: `0.1`
	- `--shadow-length--0`: `0.5px 1px 2px 0px`
	- `--shadow-length--1`: `0.5px 1px 4px 0px`
	- `--shadow-length--2`: `1px 2px 8px 0px`
	- `--shadow-length--3`: `2px 4px 16px 0px`
	- `--shadow-length--4`: `4px 8px 32px 0px`
